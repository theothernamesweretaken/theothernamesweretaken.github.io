How to install the Voting List Script

  1. Install the TamperMonkey browser add-on for Firefox (Desktop or Android), or for Chrome (Desktop only)
  2. Install the script by Clicking here

How does it look when installed?

Step 1: View the voting list for the current round
Step 2: Click the button to copy the voting list
Step 3: Paste the color-coded list wherever you want
